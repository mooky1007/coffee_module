1

Cride

크리드 시스템은 국내 최초 전문직 그룹을 위한
컨설팅 프로그램으로 제작 되었습니다.
초회 상담부터 컨설팅 제안까지 폭넓게 활용이 가능합니다.

2

API

국내 최대 규모의 KODATA(한국평가데이터)와 제휴하여
신속하고 적시성 있는 기업의 주요 원천 데이터를
실시간으로 확보하여,
다양한 컨설팅 모듈에 적용됩니다.

3

Developer Docs

전문직 맞춤형 컨설팅 프로그램으로 설계되어 법인 결산
데이터를 업로드하면 보다 정확한 컨설팅 시뮬레이션
모듈이 작동됩니다. 특히 업체 숫자가 많은 경우
대량 업로드 또한 가능합니다.
